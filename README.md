# Amp-Reverb-Control
This is a reverb control platform that can be placed into an ampilifer and has a remote pedal that controls the reverb effect. 

It has two parts: the master mcu unit that goes into the amplifier and a slave unit in a pedal. You replace the reverb pot with the digital one on the PCB so the audio signal stays in the amp. Then connect the exisiting reverb pot to the board. 
Connect the master mcu to the pedal via a 6 pin GX12 cable and you are now able to remotely control the reverb. You can select between two different levels and turn the effect on or off.
